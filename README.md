# Tamer Reklam CRM

Tamer Reklam is akislari icin statik CRM ve haftalik takvim prototipi.

## Ozellikler

- Sirket, kontak ve teklif aramasi
- Google Calendar benzeri haftalik takvim
- Tam gun zaman araligi, acilista 08:00-20:00 odagi
- Takvim slotundan kayit olusturma
- Mevcut takvim kaydini duzenleme
- Tabela is tipleri, urunler, potansiyel is ve ek dosya alanlari

## Yerelde calistirma

```bash
python3 -m http.server 8000
```

Sonra `http://localhost:8000` adresini acin.
